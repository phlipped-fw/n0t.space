PHLIPPED is a purple-team framework focused on modern physical electronic attacks: Flipper Zero, RubberDucky / HID injection, Raspberry Pi drop devices, sub-GHz replay, badge cloning, evil USB cables.

For every catalogued attack, three layers (**prevent**, **detect**, **respond**) mapped to MITRE ATT&CK, MITRE D3FEND, NIST CSF 2.0, ISO 27001 Annex A and NIS2.
