PHLIPPED - **Physical Layered Intrusion Prevention, Protection & Evaluation Defense** - is a defensive framework focused on physical electronic attacks that became trivially accessible in the last five years.

Frameworks dominant today (NIST CSF 2.0, ISO 27001, CIS Controls, MITRE ATT&CK) cover physical security only at a surface level. PHLIPPED specialises them with concrete detections, an auditable assessment methodology, and hardening playbooks targeted at SMEs without an internal red team.

## Outputs

- Open-source threat catalog
- Python assessment CLI (`phlipped-assess`)
- Detection rules (Wazuh + Sigma)
- Hardening playbook
- Whitepaper & conference talk
